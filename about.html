    关于本站

    把工程实践与 ACG 兴趣揉在一起的小站，专注于体验与可维护性。

    初心

    这里是我持续打磨 Web 工程能力的空间。站点采用 Next.js 15（App Router & RSC 优先）与 Tailwind CSS 4，在性能、可访问性与工程质量之间寻找平衡。

    内容以技术记录与 ACG 相关的阅读/观影笔记为主，力求简洁、稳定与可演进。

    功能亮点

    • Markdown 渲染 + 语法高亮（GFM）与代码块一键复制
    • 全局图片点击放大，可通过 data-no-zoom 排除
    • 动漫/漫画/轻小说阅读：动态路由、章节导航、本地与远程资源混合
    • TXT 自动编码识别与换行规范化，阅读体验顺滑
    • 深浅色主题切换并持久化
    • 站内搜索：统一检索文章/动漫/漫画/轻小说并可直接跳转

    技术栈

    Next.js 15（App Router / RSC）
    React 19
    Tailwind CSS 4
    React Markdown + remark-gfm
    Prism 高亮
    dayjs / nanoid

    关于我

    你好，我是 E.M。我喜欢以工程视角组织内容，也喜欢在 ACG 作品中寻找灵感。《罪恶王冠》《魔女之旅》等作品影响了我对叙事与情绪的理解。

    这个站点既是练习场，也是不断更新的作品集。如你有建议或想法，欢迎交流。
